# Prompt Builder Module

🚀 Overview

The Prompt Builder is a component of the parser module in the JIRA Code Automation project.

It is responsible for converting structured ticket data into a well-formatted prompt that can be used by the generator module to create code and test cases.

parser/
├── prompt_builder.py


# How It Works
- Reads feature name from input data
- Appends requirements as bullet points
- Appends validation rules
- Adds instruction for test case generation
- Returns final formatted prompt
